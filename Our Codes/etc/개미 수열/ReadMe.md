# 개미 수열

베르나르 베르베르의 소설 개미에서 알려진 개미 수열이 있다.

개미 수열의 예시는 다음과 같다.

```
1
11
12
1121
122111
112213
...
```

방식은 윗 줄에 있는 숫자의 수를 세는 것이다.

윗 줄의 수가 1121이라면, 1이 2개이므로 12, 2개 1개이므로 21, 1이 1개이므로 11로 122111이다.

<br>

첫 줄에 수열의 첫 번째 줄을 입력받고, 이후 줄부터 몇 번째 수열을 출력할지 입력받는다.

몇 번째 수열을 출력할지는 여러 번 입력될 수 있다.

<br>

#### 입/출력 예시

##### 입력
```
33
5
2
```

##### 출력
```
31132113
32
```
